# 💻 Arduino Uno | 4 Servo Motor Control Project

## 📖 Project Overview

This project demonstrates how to control four SG90 servo motors using an Arduino Uno in Tinkercad Circuits.

The project performs the following tasks:

- ✅ Controls four SG90 servo motors simultaneously.
- 🔄 Runs the Sweep motion for the first 2 seconds.
- 🛑 Stops all servo motors after 2 seconds.
- 🎯 Holds all servo motors at a fixed angle of 90°.

---

## 🧰 Components

- Arduino Uno
- 4 × Micro Servo SG90
- Breadboard
- Jumper Wires

---

## 🔌 Wiring

| Servo | Signal Pin |
|--------|------------|
| Servo 1 | D3 |
| Servo 2 | D5 |
| Servo 3 | D6 |
| Servo 4 | D9 |

- All servo power wires are connected to the Arduino 5V pin.
- All servo ground (GND) wires are connected to the Arduino GND pin.

## 💻 Arduino Code

The project uses the Servo library to control four servo motors.

### 🚀 Workflow

1. Initialize four servo motors.
2. Run the Sweep movement.
3. Continue sweeping for 2 seconds.
4. Stop all servo motors.
5. Hold every servo motor at 90°.
